# 🛒 Rossmann Store Sales Prediction

🚀 A Machine Learning project to predict daily sales of Rossmann stores using historical data.

---

## 📌 Overview

This project predicts store sales using advanced data analysis and machine learning techniques. It helps businesses make better decisions related to inventory, staffing, and promotions.

---

## 📊 Dataset

The dataset includes:
✔ Store ID
✔ Sales
✔ Customers
✔ Promo
✔ Holidays (State & School)
✔ Date

---

## 🛠️ Tech Stack

🐍 Python
📊 Pandas, NumPy
📈 Matplotlib
🤖 Scikit-learn

---

## ⚙️ Workflow

✔ Data Cleaning
✔ Exploratory Data Analysis (EDA)
✔ Feature Engineering
✔ Model Training
✔ Evaluation

---

## 🤖 Models Used

🔹 Linear Regression
🔹 Random Forest Regressor

---

## 📈 Results

Achieved good prediction accuracy and identified key factors affecting sales like promotions and holidays.
