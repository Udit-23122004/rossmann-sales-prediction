# 📊 Sales Forecasting using Machine Learning

## 🚀 Project Overview
This project predicts daily store sales using time-series forecasting techniques and XGBoost.

## 🔥 Key Features
- Time-based Train-Test Split
- Lag Feature Engineering
- Rolling Mean Features
- XGBoost Model
- Model Serialization using Pickle

## 📈 Model Performance
- R² Score: 0.94
- MAE: 510

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
